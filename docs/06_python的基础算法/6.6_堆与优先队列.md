# 6.6 堆与优先队列

堆（Heap）是优先级队列的标准实现——每次取出的不是"最早进入的"，而是"优先级最高的"。Python 的 `heapq` 模块提供了最小堆的全部操作。

> 📖 **官方文档**：[heapq — Heap queue algorithm](https://docs.python.org/3/library/heapq.html)

---

## 1.1 堆是什么

**堆是完全二叉树**，且满足堆性质：**父节点的值 ≤ 子节点的值**（最小堆）。

```
        1          ← 根是所有元素中最小的
      /   \
     3     2        ← 每层的值都比上一层大
    / \   /
   7   4 5          ← 但同层之间没有大小关系
```

用数组表示完全二叉树：
- 节点 i 的左子节点：`2i + 1`
- 节点 i 的右子节点：`2i + 2`
- 节点 i 的父节点：`(i - 1) // 2`

```python-run
import heapq

# heapq 提供的都是最小堆操作
heap = []

heapq.heappush(heap, 3)
heapq.heappush(heap, 1)
heapq.heappush(heap, 4)
heapq.heappush(heap, 2)

print(f"堆: {heap}")               # [1, 2, 4, 3]
print(f"heappop → {heapq.heappop(heap)}")   # 1 (最小)
print(f"heappop → {heapq.heappop(heap)}")   # 2
print(f"剩余: {heap}")             # [3, 4]
```

> `heap[0]` 永远是最小元素，不需要 `min()`。

---

## 1.2 核心操作复杂度

| 操作 | heapq 函数 | 复杂度 |
|------|-----------|--------|
| 插入 | `heappush(heap, item)` | O(log n) |
| 弹出最小 | `heappop(heap)` | O(log n) |
| 查看最小 | `heap[0]` | O(1) |
| 建堆 | `heapify(list)` | **O(n)** |
| 替换堆顶 | `heapreplace(heap, item)` | O(log n) |
| 弹出+压入 | `heappushpop(heap, item)` | O(log n) |

**`heapify` 是 O(n) 不是 O(n log n)**——自底向上的建堆过程，多数节点只需要很少的比较。

```python-run
import heapq

data = [3, 1, 4, 1, 5, 9, 2, 6]
heapq.heapify(data)                    # O(n) 原地转堆
print(f"heapify 后: {data}")

# Top-3: nlargest 内部自动选择堆/排序
print(f"最大的 3 个: {heapq.nlargest(3, data)}")
print(f"最小的 3 个: {heapq.nsmallest(3, data)}")
```

**`nlargest`/`nsmallest` 的智能选择**：
- k = 1 → `min()`/`max()`
- k ≤ n 且 n 较小 → `sorted()` 然后切片
- k 较大 → 建堆

## 1.3 最大堆的 Python 惯用法

Python 只提供了最小堆。最大堆的技巧：**把值取负数**。

```python-run
import heapq

class MaxHeap:
    def __init__(self):
        self._heap = []

    def push(self, val):
        heapq.heappush(self._heap, -val)

    def pop(self):
        return -heapq.heappop(self._heap)

    def peek(self):
        return -self._heap[0]

    def __len__(self):
        return len(self._heap)

mh = MaxHeap()
for v in [1, 5, 3, 7, 2]:
    mh.push(v)
print(f"max pop → {mh.pop()}")   # 7
print(f"max pop → {mh.pop()}")   # 5
```

## 1.4 堆的经典应用

### Top-K 问题

```python-run
import heapq

def top_k_stream(nums, k):
    """流式 Top-K：维护一个大小为 k 的最小堆"""
    heap = []
    for v in nums:
        if len(heap) < k:
            heapq.heappush(heap, v)
        elif v > heap[0]:          # 比堆里最小的还大
            heapq.heapreplace(heap, v)
    return heap

stream = [3, 1, 5, 2, 8, 1, 9, 4, 7]
print(f"Top-3: {sorted(top_k_stream(stream, 3), reverse=True)}")  # [9, 8, 7]
```

### 多路归并

```python-run
import heapq

# 合并 K 个有序序列
def merge_k_sorted(lists):
    """用堆合并 K 个有序列表，O(N log K)"""
    heap = []
    # 每个列表的第一个元素入堆：(值, 列表索引, 元素索引)
    for i, lst in enumerate(lists):
        if lst:
            heapq.heappush(heap, (lst[0], i, 0))

    result = []
    while heap:
        val, list_idx, elem_idx = heapq.heappop(heap)
        result.append(val)
        if elem_idx + 1 < len(lists[list_idx]):
            next_val = lists[list_idx][elem_idx + 1]
            heapq.heappush(heap, (next_val, list_idx, elem_idx + 1))
    return result

print(merge_k_sorted([[1, 4, 7], [2, 5, 8], [3, 6, 9]]))
# [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

### 任务调度器

```python-run
import heapq

def schedule_tasks(tasks):
    """简单任务调度：先执行高优先级的任务"""
    tasks = [(-p, name) for p, name in tasks]  # 负数 = 最大堆
    heapq.heapify(tasks)

    while tasks:
        priority, name = heapq.heappop(tasks)
        print(f"执行任务: {name} (优先级={-priority})")

schedule_tasks([(1, "日志清理"), (5, "紧急修复"), (3, "定时备份"), (4, "代码审查")])
```

## 1.5 `queue.PriorityQueue` vs `heapq`

| 维度 | `heapq` | `queue.PriorityQueue` |
|------|---------|----------------------|
| 线程安全 | ❌ 否 | ✅ 是 |
| 用途 | 算法、单线程优先队列 | 多线程任务通信 |
| 开销 | 低 | 额外锁和条件变量 |
| `get` 行为 | `heappop`（立即返回） | 阻塞等待直到有元素 |

> 算法用 `heapq`，线程通信用 `PriorityQueue`。

## 本章小结

1. **堆 = 完全二叉树 + 堆性质**（父 ≤ 子）。用数组存储，索引公式：`left=2i+1`，`right=2i+2`，`parent=(i-1)//2`
2. `heappush`/`heappop` 都是 **O(log n)**，`heap[0]` 查看最小是 **O(1)**，`heapify` 建堆是 **O(n)**
3. **最大堆用负数**：`heappush(heap, -val)`，但自定义对象可以用 `(priority, counter, item)` 三元组
4. **Top-K 维护大小为 K 的最小堆**——比堆顶小的忽略，比堆顶大的替换
5. **多路归并** 用堆合并 K 个有序序列，O(N log K)；`heapq.merge` 已实现（基于堆的归并）
