# 6.5 树与二叉树

树是非线性数据结构的核心——从文件系统到 HTML DOM，从数据库索引到编译器的 AST，树的变体无处不在。Python 没有内置的树类型，但用类来表达树结构非常自然。

---

## 1.1 树的基本概念

```
        A (根节点 root)
       / \
      B   C       ← B, C 是 A 的子节点
     / \   \
    D   E   F     ← D, E, F 是叶子节点（没有子节点）
```

| 术语 | 定义 |
|------|------|
| 根（Root） | 树的顶端节点，没有父节点 |
| 叶子（Leaf） | 没有子节点的节点 |
| 深度（Depth） | 从根到该节点的边数 |
| 高度（Height） | 从该节点到最深叶子的边数 |
| 度（Degree） | 一个节点的子节点数量 |
| 二叉树 | 每个节点最多有 2 个子节点 |

## 1.2 二叉树的 Python 表达

```python-run
from __future__ import annotations
from typing import Generic, TypeVar

T = TypeVar('T')

class TreeNode(Generic[T]):
    def __init__(self, val: T, left: TreeNode[T] | None = None,
                 right: TreeNode[T] | None = None):
        self.val = val
        self.left = left
        self.right = right

# 构建:
#     1
#    / \
#   2   3
#  / \
# 4   5
tree = TreeNode(1,
    TreeNode(2, TreeNode(4), TreeNode(5)),
    TreeNode(3))
```

### 二叉树的三种遍历

遍历是理解"递归思想"的最佳入口。三种遍历的区别只在 **访问根节点的时机**。

```python-run
# 前序：根 → 左 → 右  （序列化用）
def preorder(root):
    if not root: return
    print(root.val, end=' ')
    preorder(root.left)
    preorder(root.right)

# 中序：左 → 根 → 右  （BST 有序输出）
def inorder(root):
    if not root: return
    inorder(root.left)
    print(root.val, end=' ')
    inorder(root.right)

# 后序：左 → 右 → 根  （先处理子节点再处理父节点）
def postorder(root):
    if not root: return
    postorder(root.left)
    postorder(root.right)
    print(root.val, end=' ')

# 层序：按层从上到下，从左到右（BFS）
from collections import deque

def levelorder(root):
    if not root: return
    q = deque([root])
    while q:
        node = q.popleft()
        print(node.val, end=' ')
        if node.left:  q.append(node.left)
        if node.right: q.append(node.right)

# 构建示例树
class TreeNode:
    def __init__(self, v, l=None, r=None):
        self.val, self.left, self.right = v, l, r

t = TreeNode(1, TreeNode(2, TreeNode(4), TreeNode(5)), TreeNode(3))
print("前序:", end=' '); preorder(t); print()
print("中序:", end=' '); inorder(t); print()
print("后序:", end=' '); postorder(t); print()
print("层序:", end=' '); levelorder(t); print()
```

## 1.3 二叉搜索树（BST）

**BST 的性质**：左子树所有值 < 根 < 右子树所有值。利用这个性质，查找、插入、删除都是 O(h)，h 是树高。

```python-run
class BST:
    def __init__(self):
        self.root = None

    def insert(self, val):
        """迭代插入，O(h)"""
        node = self._make_node(val)
        if not self.root:
            self.root = node
            return
        cur = self.root
        while True:
            if val < cur.val:
                if not cur.left:
                    cur.left = node
                    return
                cur = cur.left
            else:
                if not cur.right:
                    cur.right = node
                    return
                cur = cur.right

    def search(self, val):
        """二分查找，O(h)"""
        cur = self.root
        while cur:
            if val == cur.val:
                return True
            cur = cur.left if val < cur.val else cur.right
        return False

    def _make_node(self, val):
        t = type('Node', (), {})
        t.val, t.left, t.right = val, None, None
        return t

    def inorder(self):
        result = []
        def dfs(node):
            if node:
                dfs(node.left)
                result.append(node.val)
                dfs(node.right)
        dfs(self.root)
        return result

# 演示
bst = BST()
for v in [5, 3, 7, 1, 4, 6, 8]:
    bst.insert(v)

print(f"中序遍历（有序）: {bst.inorder()}")
print(f"search(4): {bst.search(4)}, search(9): {bst.search(9)}")
```

### BST 的退化问题

最坏情况（按顺序插入 1, 2, 3, 4, 5）→ 退化成链表 → O(n)。这就是为什么需要**平衡二叉搜索树**（AVL、红黑树）：

> Python 没有内置 AVL/红黑树。但标准库 `bisect` 模块用有序 list 模拟了类似功能——查找 O(log n)，插入 O(n)。需要"有序 + 动态插入"直接用 `heapq` 或第三方库 `sortedcontainers`。

## 1.4 树的常见变体速查

| 结构 | 特点 | Python 工具 |
|------|------|------------|
| **堆（Heap）** | 完全二叉树，父 ≤ 子 | `heapq`（见 6.6 节） |
| **Trie（前缀树）** | 按字符前缀组织，高效前缀搜索 | 手写（见 6.12 节） |

| **线段树** | 区间查询 O(log n) | 手写 |
| **B-Tree** | 数据库索引用（节点存多个 key） | — |

## 1.5 树问题的两个通用思路

### 1. 递归（分治）— 90% 的树问题

```python
def tree_depth(root):
    if not root:
        return 0
    return 1 + max(tree_depth(root.left), tree_depth(root.right))
```

### 2. 层序遍历（BFS）— 按层处理

```python
def level_order(root):
    from collections import deque
    q = deque([root])
    while q:
        node = q.popleft()
```

## 本章小结

1. **树 = 非线性、有层级关系**。二叉树每个节点最多两个子节点
2. **三种深度遍历**：前序（根→左→右）、中序（左→根→右，BST 有序输出）、后序（左→右→根）
3. **层序遍历 = BFS**：用队列逐层处理
4. **BST 查找 O(log n)，但退化后 O(n)**——平衡二叉树（AVL/红黑树）解决退化问题
5. Python 没有内置树——用类定义 + 递归处理，代码简洁且清晰
