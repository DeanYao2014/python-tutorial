# 6.9 排序算法

排序是理解"算法内功"的最佳入口——不同策略（比较、分治、非比较）在你眼前展开。Python 的 `sorted()` 和 `list.sort()` 内部使用 Timsort（最高效的比较排序），但理解经典排序的思想才是算法思维的基础。

> 📖 **官方文档**：[Sorting HOW TO](https://docs.python.org/3/howto/sorting.html), [Timsort](https://en.wikipedia.org/wiki/Timsort)

---

## 1.1 Timsort：Python 的生产级排序

`sorted()` 和 `list.sort()` 内部都是 **Timsort**：归并排序 + 插入排序的混合体。稳定、O(n log n)、对现实数据（部分有序）有极佳的适应性。

```python-run
# sorted() 基本用法
nums = [3, 1, 4, 1, 5, 9, 2, 6]
print(f"sorted（新列表）: {sorted(nums)}")
print(f"原列表未变: {nums}")

nums.sort()           # 原地排序，只适用于 list
print(f"原地排序后: {nums}")

# key 参数：排序前先对每个元素调用 key 函数
words = ["banana", "apple", "Cherry", "date"]
print(f"按长度: {sorted(words, key=len)}")
print(f"不区分大小写: {sorted(words, key=str.lower)}")
```

### `key` 函数的本质：Schwartzian Transform（装饰-排序-去装饰）

```python-run
# sorted(data, key=f) 等价于
# 1. 装饰: [(f(x), x) for x in data]
# 2. 排序: 按元组第一个元素
# 3. 去装饰: 取出原元素

# 示例：按成绩排序（降序）
students = [
    ("张三", 85), ("李四", 92), ("王五", 78), ("赵六", 92)
]
# 按成绩降序，同名次按姓名升序（Python 的 sort 是稳定的）
result = sorted(students, key=lambda s: (-s[1], s[0]))
print(f"成绩排名: {result}")
```

## 1.2 经典排序算法对比

```python-run
# ── 冒泡排序 O(n²) —— 相邻交换，大的"浮"上去 ──
def bubble_sort(arr):
    a = arr.copy()
    n = len(a)
    for i in range(n - 1):
        swapped = False
        for j in range(n - 1 - i):
            if a[j] > a[j + 1]:
                a[j], a[j + 1] = a[j + 1], a[j]
                swapped = True
        if not swapped:       # 提前终止优化
            break
    return a

# ── 选择排序 O(n²) —— 每次选最小的放前面 ──
def selection_sort(arr):
    a = arr.copy()
    n = len(a)
    for i in range(n - 1):
        min_idx = i
        for j in range(i + 1, n):
            if a[j] < a[min_idx]:
                min_idx = j
        a[i], a[min_idx] = a[min_idx], a[i]
    return a

# ── 插入排序 O(n²)，但对"几乎有序"的数据是 O(n) ──
def insertion_sort(arr):
    a = arr.copy()
    for i in range(1, len(a)):
        key = a[i]
        j = i - 1
        while j >= 0 and a[j] > key:
            a[j + 1] = a[j]
            j -= 1
        a[j + 1] = key
    return a

data = [5, 2, 4, 6, 1, 3]
print(f"冒泡: {bubble_sort(data)}")
print(f"选择: {selection_sort(data)}")
print(f"插入: {insertion_sort(data)}")
```

### 各类排序一张表

| 算法 | 最好 | 平均 | 最坏 | 空间 | 稳定 | 核心思想 |
|------|------|------|------|------|------|---------|
| **冒泡** | O(n) | O(n²) | O(n²) | O(1) | ✅ | 相邻交换 |
| **选择** | O(n²) | O(n²) | O(n²) | O(1) | ❌ | 每次找最小 |
| **插入** | O(n) | O(n²) | O(n²) | O(1) | ✅ | 像理扑克牌 |
| **归并** | O(n log n) | O(n log n) | O(n log n) | O(n) | ✅ | 分治+合并 |
| **快速** | O(n log n) | O(n log n) | O(n²) | O(log n) | ❌ | 分治+分区 |
| **堆排序** | O(n log n) | O(n log n) | O(n log n) | O(1) | ❌ | 堆维护 |
| **Timsort** | O(n) | O(n log n) | O(n log n) | O(n) | ✅ | 归并+插入 |

## 1.3 排序的 Python 进阶用法

```python-run
from operator import itemgetter, attrgetter
from functools import cmp_to_key

# itemgetter：比 lambda 更快
students = [("张三", 85), ("李四", 92), ("王五", 78)]
print(f"按成绩: {sorted(students, key=itemgetter(1))}")

# 多级排序：稳定的 sort，多次调用
students.sort(key=itemgetter(0))     # 先按姓名
students.sort(key=itemgetter(1))     # 再按成绩，姓名相同时保持原来顺序

# cmp_to_key：把老式 cmp 函数转为 key 函数
def cmp_abs(a, b):
    """按绝对值排序"""
    return abs(a) - abs(b)

nums = [-5, 3, -1, 4, -2]
print(f"按绝对值: {sorted(nums, key=cmp_to_key(cmp_abs))}")
```

## 1.4 桶排序与计数排序

当数据范围有限时，**非比较排序**可以达到 O(n)：

```python-run
# 计数排序：数据范围 [0, k)，O(n+k)，不是比较排序
def counting_sort(arr, k=100):
    count = [0] * k
    for v in arr:
        count[v] += 1
    result = []
    for v in range(k):
        result.extend([v] * count[v])
    return result

# 桶排序：把数据分散到桶里，每个桶内排序
def bucket_sort(arr, bucket_size=5):
    if not arr:
        return arr
    min_v, max_v = min(arr), max(arr)
    bucket_count = (max_v - min_v) // bucket_size + 1
    buckets = [[] for _ in range(bucket_count)]

    for v in arr:
        buckets[(v - min_v) // bucket_size].append(v)

    result = []
    for bucket in buckets:
        result.extend(sorted(bucket))  # 桶内用比较排序
    return result

data = [29, 25, 3, 49, 9, 37, 21, 43]
print(f"计数排序: {counting_sort([3, 1, 4, 1, 5, 3], 100)}")
print(f"桶排序: {bucket_sort(data)}")
```

## 本章小结

1. **生产代码用 `sorted()`/`list.sort()`（Timsort）**，稳定、O(n log n)、对部分有序数据自适应 O(n)
2. **`key` 参数是 Schwartzian Transform**（装饰-排序-去装饰），比 `cmp` 快且更声明式
3. **插入排序对"几乎有序"数据是 O(n)**——这就是为什么 Timsort 用它合并小 run
4. **归并排序稳定的 O(n log n) 但 O(n) 空间**，快速排序原地但最坏 O(n²)——TimSort 取两者之长
5. **非比较排序（计数、桶）可到 O(n)**，但只适用于数据范围有限的场景
