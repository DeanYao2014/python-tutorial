# 0.2 字符串与编码

## 创建字符串

Python 没有"字符"类型——单个字符也是长度为 1 的字符串。单引号和双引号没有区别：

```python-run
s1 = 'hello'
s2 = "hello"
print(s1 == s2)  # True — 一样的

# 三引号用于多行文本
poem = """春眠不觉晓，
处处闻啼鸟。"""
print(poem)
```

原始字符串 `r"..."` 让反斜杠保持字面含义，正则表达式和 Windows 路径必备：

```python-run
print(r"C:\Users\names")   # \n 不会被当成换行
```

## f-string：Python 最强大的字符串工具

f-string（格式化字符串字面量，Formatted String Literal）在 `{}` 里可以写**任意 Python 表达式**：

```python-run
name, age = "小明", 10
print(f"{name} 明年 {age + 1} 岁")

from math import pi
print(f"π ≈ {pi:.3f}")

# Python 3.8+：调试专用语法 {变量=}
print(f"{name = }, {age = }")
```

Python 3.12 起，f-string 内部可以使用和外部一样的引号：

```python-run
d = {"lang": "Python", "ver": 3.12}
# Python 3.12 起支持嵌套引号：print(f"{d["lang"]}")
print(f"语言: {d['lang']}")    # 3.11 兼容写法
```

## 字符串常用方法

```python-run
# split：按分隔符拆成列表
"a,b,c".split(",")

# join：把列表粘成字符串
",".join(["a", "b", "c"])

# strip / replace / startswith
"  hi  ".strip()
"hello".replace("h", "H")
"hello".startswith("he")
```

## 字符编码：一句话理解

**Unicode** 是"字符到编号"的映射表（`'中'` → `U+4E2D`），**UTF-8** 是"编号到字节"的编码规则。`encode()` 把字符串变成字节，`decode()` 把字节变回字符串：
> ASCII编码是一个字节,只能表示0-255; `A` 的十进制是`65`, 二进制`01000001`;
> 
> Unicode是为了解决ASCII编码范围不够, 通常Unicode编码是2个字节;
> 
> UTF-8: 可变长编码,节约空间; 
> 在计算机内存中，统一使用Unicode编码，当需要保存到硬盘或者需要传输的时候，就转换为UTF-8编码。
```python-run
text = "Python 🐍"
b = text.encode("utf-8")    # str → bytes
print(f"编码: {b}")
print(f"解码: {b.decode('utf-8')}")  # bytes → str
```

> **关键规则**：str 是给人看的（字符），bytes 是给机器存的（字节）。读写文件、网络传输都用 bytes。到了 IO 边界才做编解码。

## 本章小结

- Python 没有"字符"类型，单字符也是长度为 1 的字符串；单引号和双引号等价，三引号用于多行文本
- f-string 在 `{}` 内可以写任意表达式，`{变量=}` 调试语法（3.8+），3.12 起支持嵌套引号
- 核心字符串方法：`split` 拆分、`join` 拼接、`strip` 去空白、`replace` 替换、`startswith`/`endswith` 检查前后缀
- Unicode 是"字符到编号"的映射，UTF-8 是"编号到字节"的编码规则；`encode()` str→bytes，`decode()` bytes→str——IO 边界才做编解码
