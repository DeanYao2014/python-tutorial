# Python重要的知识点

## Python的作用域贵则LEGB规则

Python的变量查找顺序
1. local本地 → 当前函数内部
2. Enclosing (外层函数) → 嵌套函数的外层
3. Global(全局) → 整个文件顶层
4. Built-in(内置) → Python自带的(print,len等)

## 闭包

**闭包 = 内部函数 + 保留了对外部函数变量的引用,保存在closure元组中**不是复制值，是引用（指针），外部函数结束了，这个引用依然存在。

`__closure__` 就是 Python 用来存储闭包引用的外部变量的魔术属性！
1. 这是一个元组
2. 里面保存的是cell对象
3. 每个cell保存一个被引用的外部变量
```python-run
def outer(x):
    # 外部变量
    y = 100

    def inner():
        # 内层引用外部 x、y → 形成闭包
        return x + y

    return inner  # 返回内层函数

# 创建闭包实例
func = outer(10)

# 查看闭包
print(func.__closure__)
# 输出：(<cell at 0x...: int object at 0x...>, <cell at 0x...: int object at 0x...>)

# 查看变量内容
for cell in func.__closure__:
    print(cell.cell_contents)  # 输出 10 和 100
```




## nonlocal 和 global

`nonlocal`: 声明指向外层函数的变量
`global` : 声明为全集的变量

```python
def test():
    # 局部变量x
    x = 200 
    print(x)

def test():
    # x为全局变量
    global x
    x = 100
    print(x)


def outer():
    a = 10
    
    def inner():
        nonlocal a  # 声明用外层函数的变量
        a = 20
    
    inner()
    print(a)  # 20
```

## python 变量

1. python的变量不用声明,`第一次赋值` = 创建
2. **只有函数,类会创建新作用域**
3. `if/for/while/with/try` 都不会创建作用域
4. 内部想要改变外部变量: 使用global/nonlocal
```python-run
'for循环不会创建新的作用域, i其实是全局变量, 最后打印i=8'
for i in range(1,9):
  pass

print(f"for之外访问i: {i}")
```
