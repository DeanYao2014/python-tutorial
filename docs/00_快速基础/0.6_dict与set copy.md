# 0.6 dict 与 set

## dict（字典）—— Python 的基石

dict 是 Python 内部实现的基础——对象属性、模块命名空间、关键字参数都用 dict。理解 dict 对理解 Python 至关重要。

```python-run
person = {"name": "张三", "age": 25, "city": "北京"}
print(person["name"])

# get：安全取值，key 不存在返回默认值而不是报错
print(person.get("gender", "未知"))
```

### 增删改查

```python-run
d = {"a": 1}
d["b"] = 2          # 新增
d["a"] = 99         # 修改
popped = d.pop("b") # 删除并返回
print(d, popped)
```

### 遍历 dict

```python-run
d = {"x": 10, "y": 20}
for key in d:               # 默认遍历键
    print(key, end=" ")

for key, value in d.items():  # 同时拿键值
    print(f"{key}={value}", end=" ")
```

Python 3.7+ dict 保持**插入顺序**，所以遍历顺序和添加顺序一致。

### 处理缺失键

```python-run
# get 带默认值 → 最简洁的计数器
counts = {}
for c in "aabbbcccc":
    counts[c] = counts.get(c, 0) + 1
print(counts)
```

`collections.defaultdict` 更进一步：

```python
from collections import defaultdict
d = defaultdict(list)
d["a"].append(1)   # 自动为缺失 key 创建空 list
```

### dict comprehension（字典推导式）

```python-run
squares = {x: x**2 for x in range(6)}
print(squares)  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

## set（集合）—— 数学里的集合

set 三大特性：**无重复、无序、O(1) 成员检查**。

```python-run
# 去重——set 最常见的用法
nums = [3, 1, 2, 3, 1, 4]
print(f"去重: {set(nums)}")
```

### 集合运算

```python-run
a, b = {1, 2, 3}, {2, 3, 4}
print(f"a & b  交集: {a & b}")
print(f"a | b  并集: {a | b}")
print(f"a - b  差集: {a - b}")
print(f"a ^ b  对称差: {a ^ b}")
```

### 成员检查 —— 为什么用 set 而不是 list

```python-run
# 如果经常做"是否包含"检查，用 set
big_set = set(range(10_000))
# "x in big_set" 是 O(1) — 哈希查找
# "x in big_list" 是 O(n) — 逐个比较
```

> **规则**：需要去重、集合运算、频繁成员检查时用 `set`。需要顺序和索引时用 `list`。

## 本章小结

- `dict` 是 Python 内部基石——对象属性、命名空间、关键字参数都基于它；Python 3.7+ 保持插入顺序
- `get(key, default)` 安全取值，`defaultdict` 自动为缺失键创建默认值——处理缺失键的两种常用手段
- `set` 三大特性：无重复、无序、O(1) 成员检查；常用于去重、集合运算和频繁成员判断
- 集合运算：`&` 交集、`|` 并集、`-` 差集、`^` 对称差；`in` 检查 set 为 O(1) vs list 为 O(n)
