# 0.4 条件判断与模式匹配

## `if / elif / else`

Python 用缩进表示代码块，不需要 `{}` 或 `begin/end`：

```python-run
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

### Python 特色：链式比较

```python-run
x = 5
print(3 < x < 10)    # True，等价于 3 < x and x < 10
print(3 < x >= 7)    # False
```

## 模式匹配 `match case`（Python 3.10+）

`match` 不只是"加强版 switch"——它是一个**解构 + 匹配 + 绑定**三合一的表达式。

### 字面量匹配

```python-run
def http_desc(code):
    match code:
        case 200: return "成功"
        case 301: return "重定向"
        case 404: return "未找到"
        case 500: return "服务器错误"
        case _:   return "未知状态"

print(http_desc(200))
```

`_` 是**通配符**，匹配任何值，一般放在最后。

### 序列匹配 —— 自动解构

```python-run
def analyze_list(lst):
    match lst:
        case []:
            return "空列表"
        case [x]:
            return f"只有一个元素: {x}"
        case [first, *rest]:
            return f"开头: {first}, 后面还有 {len(rest)} 个"
        case [*_, last]:
            return f"末尾: {last}"

print(analyze_list([1, 2, 3, 4]))
```

### 守卫（Guard）—— 匹配 + 条件

```python-run
def describe(n):
    match n:
        case 0:            return "零"
        case n if n > 0:   return "正数"
        case n if n < 0:   return "负数"

print(describe(42))
```

## `if` vs `match` 的选择

| 场景 | 推荐 |
|------|------|
| 单条件判断 | `if` |
| 多条件 + 需要提取字段 | `match` |
| 简单值比较 | 都可以，`match` 更紧凑 |
| 需要 `and`/`or` 组合条件 | `if` |

> `match` 的真正威力在进阶章节（类模式、映射模式、OR 模式）中展开。

## 本章小结

- `if / elif / else` 用缩进表示代码块，Python 支持链式比较如 `3 < x < 10`
- `match case`（3.10+）是解构 + 匹配 + 绑定三合一的表达式，`_` 是匹配任意值的通配符
- `match` 支持序列模式自动解构（如 `[first, *rest]`）和守卫条件（`case n if n > 0`）
- 单条件判断用 `if`，多条件且需要提取/解构数据字段时用 `match`
