# 0.5 循环

## `for` — Python 最常用的循环

`for` 遍历**任何可迭代对象**——list、tuple、字符串、dict、文件，甚至是自己定义的对象：

```python-run
for char in "Python":
    print(char, end=" ")
```

### `range` — 生成数字序列

```python-run
print(list(range(5)))          # [0, 1, 2, 3, 4]
print(list(range(2, 6)))       # [2, 3, 4, 5]
print(list(range(10, 0, -2)))  # [10, 8, 6, 4, 2]
```

> `range` 是"惰性"的——不占内存，只在迭代时生成下一个值。

## 三个必备搭档

### `enumerate` — 同时拿索引和值

```python-run
fruits = ["苹果", "香蕉", "橘子"]
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}. {fruit}")
```

### `zip` — 并行迭代

```python-run
names = ["张三", "李四", "王五"]
scores = [85, 92, 78]

for name, score in zip(names, scores):
    print(f"{name}: {score}")
```

`zip` 按**最短**序列停止。需要按最长时用 `itertools.zip_longest`。

### 循环的 `else` 子句

Python 的 `for`/`while` 可以跟 `else`：当循环**正常结束**（没有遇到 `break`）时执行：

```python-run
for n in range(5):
    if n == 10:
        print("找到了")
        break
else:
    print("循环正常结束，没有 break")
```

## `while` — 条件循环

```python-run
n = 3
while n > 0:
    print(n)
    n -= 1
print("出发！")
```

`while` 适合"不知道要循环多少次"的场景。

## 推导式（Comprehension）—— 一行搞定循环

推导式是 Python 最优雅的特性之一：

```python-run
# 普通 for 循环
squares = []
for x in range(10):
    squares.append(x ** 2)

# 推导式 —— 一行等价
squares = [x**2 for x in range(10)]
print(squares)

# 带过滤
evens = [x for x in range(20) if x % 2 == 0]
print(evens)
```

> **规则**：简单的循环用推导式，复杂的（嵌套、多条件）保持 `for`——可读性优先。

## 本章小结

- `for` 遍历任何可迭代对象；`range` 是惰性数字序列，不占内存，只在迭代时生成值
- 三大必备搭档：`enumerate` 同时获取索引和值，`zip` 并行迭代多个序列（按最短停止），`else` 子句在循环未 `break` 时执行
- `while` 适合"不知道循环多少次"的场景——只要条件为真就继续
- 推导式（Comprehension）一行完成循环 + 过滤 + 构造，简单场景优先使用，复杂逻辑保持普通 `for`
